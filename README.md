# Megababe Progress Tracker

A React web application for tracking Lagree fitness class attendance.

## About

Megababe Progress Tracker helps Lagree enthusiasts log their classes, track their total attendance, and maintain a weekly streak to stay motivated and consistent.

## Features

- Log Lagree classes by date with optional notes
- View total classes completed
- Track your current weekly streak
- Delete entries
- Data persists with localStorage

## Tech Stack

- React
- Vite
- CSS
- localStorage

## How to Run

```bash
npm install
npm run dev
```
